 # Práctica 3                         
 
 #### MANUAL DE USUARIO "BREAKOUT"
 
 * OBJETIVO: impedir que la bola caiga. Eliminar gran número de ladrillos.
 * Al abrir el fichero, aparecerá el título "BREAKOUT" en color rojo y el canvas (1920x1080) con fondo negro.
 * INICIO: para iniciar el juego, es necesario pulsar la tecla "Espacio". De esta manera se dibuja el escenario y la bola en movimiento.
 * MOVIMIENTO PALA: mover hacia la derecha (pulsar flecha derecha del teclado); mover hacia la izquierda (pulsar flecha izquierda del teclado).
 * PUNTUACIÓN: cada ladrillo eliminado al tocar la bola suma +1 punto. La puntuación se muestra en la zona superior izquierda (color azul).
 * VIDAS: se dispone de 3 vidas en total. Cada vez que la bola cae (no se golpea con la pala), se resta -1 vida. Las vidas se muestran en la zona superior derecha (color rojo). Una vez que se han eliminado las 3 vidas se muestra un mensaje de "FIN DEL JUEGO" y el juego vuelve a INICIO (para comenzar otra vez pulsar la tecla "Espacio").
