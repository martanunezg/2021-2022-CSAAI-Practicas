# Manual Usuario Calculadora 

Calculadora realizada por Marta Núñez García.

Funciones: 

* Introducción de dígitos (0-9)
* Operación suma (+)
* Operación resta (-)
* Operación multiplicación (x)
* Operación división (/)
* Separador de números decimales (.)
* Obtención del resultado de la operación (=)
* Display en la parte superior
* Reiniciar display (AC)
* Eliminar dígito (DEL)
